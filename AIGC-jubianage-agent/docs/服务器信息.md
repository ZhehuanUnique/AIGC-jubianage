## 🔗 域名解析配置

### 步骤1：登录域名管理后台

1. 登录你的域名注册商（如：腾讯云、阿里云、新网等）
2. 进入域名管理 → DNS解析

### 步骤2：添加解析记录

在域名 `jubianai.cn` 下添加以下记录：

#### A 记录（主域名）
```
记录类型：A
主机记录：@（或留空）
记录值：119.45.121.152
TTL：600（或默认）
```

#### A 记录（www子域名）
```
记录类型：A
主机记录：www
记录值：119.45.121.152
TTL：600（或默认）
```

#### 可选：API 子域名（推荐）
```
记录类型：A
主机记录：api
记录值：119.45.121.152
TTL：600（或默认）
```

### 步骤3：等待解析生效

- 解析生效时间：通常 5-30 分钟
- 可以通过 `ping jubianai.cn` 测试是否生效
- 如果 ping 返回 `119.45.121.152`，说明解析成功
