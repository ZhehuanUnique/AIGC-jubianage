# 腾讯云购买配置推荐

## 📋 根据你的截图，推荐配置如下：

### ✅ 推荐配置

#### 1. **地域选择**
- ✅ **选择：南京**（有"惠"标签，有优惠）
- 或者选择：**广州**、**北京**（根据你的用户主要分布选择）
- ⚠️ **注意**：地域选择后无法更改，选择离用户最近的地域

#### 2. **实例配置** ⭐ 重要
- ❌ **不选**：基础配置(2核2GB) - 内存太小，可能不够用
- ✅ **推荐**：**普及配置(2核4GB)** - 性价比高，满足初期需求
- 💡 **备选**：专业配置(4核8GB) - 如果预算充足，可以直接选这个

**理由：**
- 2核2GB 内存太小，运行 Node.js + Nginx 可能不够
- 2核4GB 是性价比最高的选择，可以随时升级
- 4核8GB 如果预算允许，可以避免后续升级

#### 3. **操作系统** ⭐ 重要
- ❌ **不选**：TencentOS Server 3.3（虽然推荐，但文档和社区支持较少）
- ❌ **不选**：CentOS 7.9（已停服，2024年6月30日停止维护）
- ✅ **推荐**：**Ubuntu Server 22.04 LTS 64位**

**理由：**
- Ubuntu 22.04 LTS 维护到 2027 年
- 文档和社区支持最好
- 软件包更新快，安装方便
- 适合个人和中小型项目

#### 4. **时长选择**
- ✅ **推荐**：**半年**（5折优惠，¥35.10/月）
- 或者：**1年**（如果有更大折扣）
- 💡 **注意**：新用户首次购买有优惠，建议选择半年或1年

#### 5. **数量**
- ✅ **选择：1台**（初期1台足够）

---

### ⚙️ 第二张图的配置

#### 1. **公网IP**
- ✅ **必须勾选**：免费分配独立公网IP
- 没有公网IP无法从外网访问

#### 2. **带宽** ⭐ 重要
- ❌ **不选**：1Mbps（太小，访问会很慢）
- ✅ **推荐**：**至少 3Mbps**，建议 **5Mbps**
- 💡 **理由**：
  - 1Mbps = 128KB/s，访问速度很慢
  - 3Mbps = 384KB/s，基本够用
  - 5Mbps = 640KB/s，推荐选择

#### 3. **登录方式**
- ✅ **选择**：自动生成密码 或 设置密码
- 💡 **建议**：选择"自动生成密码"，密码会发到邮箱，更安全

#### 4. **默认配置**
- ✅ **点击查看**：确认可用区、安全组等配置
- 确保安全组开放端口：22（SSH）、80（HTTP）、443（HTTPS）

#### 5. **自动续费**
- ⚠️ **建议不勾选**：初期先不勾选，到期前手动续费
- 如果确定长期使用，可以勾选

#### 6. **协议**
- ✅ **必须勾选**：我已阅读并同意相关协议

## ❓ 常见问题

### Q: 2核2GB 够用吗？
A: 不够。运行 Node.js + Nginx + 数据库连接，至少需要 2核4GB。

### Q: 带宽选多少合适？
A: 至少 3Mbps，推荐 5Mbps。1Mbps 访问会很慢。

### Q: 可以后续升级吗？
A: 可以。随时可以在控制台升级配置，按剩余时间补差价。

### Q: Ubuntu 和 TencentOS 哪个好？
A: 推荐 Ubuntu 22.04 LTS，文档和社区支持更好。
