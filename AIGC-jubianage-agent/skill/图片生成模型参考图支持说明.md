# 图片生成模型参考图（垫图）支持说明

## 支持参考图的模型

以下模型**支持**参考图（图生图模式），当您关联了角色、场景、物品或姿势时，会自动使用参考图：

1. **nano-banana-pro** ✅
   - 支持单张参考图
   - 使用图生图模式时，生成的图片会参考您提供的图片

2. **seedream-4-0** ✅
   - 支持单张或多张参考图（最多10张）
   - 使用图生图模式时，生成的图片会参考您提供的图片

3. **seedream-4-5** ✅
   - 支持单张或多张参考图（最多10张）
   - 使用图生图模式时，生成的图片会参考您提供的图片

4. **flux-2-max** ✅
   - 支持单张或多张参考图（最多8张）
   - 使用图生图模式时，生成的图片会参考您提供的图片

5. **flux-2-flex** ✅
   - 支持单张或多张参考图（最多8张）
   - 使用图生图模式时，生成的图片会参考您提供的图片

6. **flux-2-pro** ✅
   - 支持单张或多张参考图（最多8张）
   - 使用图生图模式时，生成的图片会参考您提供的图片

## 不支持参考图的模型

以下模型**不支持**参考图（图生图模式），即使您关联了角色、场景、物品或姿势，也不会使用参考图：

1. **midjourney-v7-t2i** ❌
   - 不支持参考图（图生图模式）
   - 只能使用文生图模式
   - 注意：Midjourney 会生成4张图片的网格，然后需要手动选择放大（Upscale）

## 使用说明

1. **自动启用参考图模式**：
   - 当您在分镜管理中关联了角色、场景、物品或姿势中的任意一个时，系统会自动启用参考图模式
   - 系统会收集所有关联的图片，并传递给支持参考图的模型
   - **重要**：如果有关联图片，Midjourney v7 选项会被自动隐藏（因为 Midjourney 不支持参考图）

2. **参考图优先级**：
   - 如果关联了多个资源（角色、场景、物品、姿势），系统会收集所有图片
   - **nano-banana-pro**：只支持单张参考图，会使用第一张图片（优先级：角色 > 场景 > 物品 > 姿势）
   - **Seedream、Flux**：支持多张参考图（最多8-10张），会传递所有关联的图片

3. **图片比例**：
   - 所有分镜都会使用全局设置的图片比例（16:9、9:16 或 1:1）
   - 修改图片比例时，所有分镜都会同步更新

4. **Midjourney v7 特殊说明**：
   - Midjourney v7 数量固定为 4，不可修改
   - Midjourney 会生成4张图片的网格（2x2布局）
   - **自动 Upscale**：网格图生成完成后，系统会自动对 4 张图进行 Upscale（放大到选择的尺寸和分辨率）
   - 4 张放大后的图片会分别保存为 1、2、3、4 号图片
   - 如果有关联图片（角色、场景、物品、姿势），Midjourney v7 选项会被隐藏

## 注意事项

- 如果使用了不支持参考图的模型（如 Midjourney），系统会显示警告信息，但不会阻止生成
- 参考图的质量会影响生成结果，建议使用清晰、高质量的参考图
- 参考图的内容应该与提示词描述的内容相关，以获得最佳效果
