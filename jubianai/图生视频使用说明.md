# 即梦AI图生视频使用说明

## 功能概述

已实现即梦AI视频生成3.0 720P的图生视频功能，支持两种模式：

1. **首帧模式**：使用一张图片生成视频
2. **首尾帧模式**：使用两张图片（首帧和尾帧）生成视频

## API 文档参考

- [首帧接口文档](https://www.volcengine.com/docs/85621/1785204?lang=zh)
- [首尾帧接口文档](https://www.volcengine.com/docs/85621/1791184?lang=zh)

## 环境变量配置

### Vercel 环境变量

在 Vercel Dashboard 中配置以下环境变量：

```
JIMENG_ACCESS_KEY_ID=你的即梦AccessKeyId
JIMENG_SECRET_ACCESS_KEY=你的即梦SecretAccessKey
JIMENG_API_ENDPOINT=https://visual.volcengineapi.com
JIMENG_REGION=cn-north-1
JIMENG_SERVICE=cv
```

### Streamlit Cloud 环境变量

在 Streamlit Cloud 的 Secrets 中配置：

```toml
[secrets]
JIMENG_ACCESS_KEY_ID = "你的即梦AccessKeyId"
JIMENG_SECRET_ACCESS_KEY = "你的即梦SecretAccessKey"
JIMENG_API_ENDPOINT = "https://visual.volcengineapi.com"
JIMENG_REGION = "cn-north-1"
JIMENG_SERVICE = "cv"
BACKEND_URL = "https://jubianai.vercel.app"
```

## 使用方式

### 前端界面使用

1. 打开 Streamlit 应用
2. 在侧边栏选择服务提供商为 "jimeng"
3. 输入即梦 API 密钥（如果未在环境变量中配置）
4. 选择生成模式：
   - **首帧模式**：上传一张图片
   - **首尾帧模式**：上传两张图片（首帧和尾帧）
5. 可选：输入提示词和负面提示词
6. 调整视频参数（时长、帧率等）
7. 点击"生成视频"按钮

### API 调用示例

#### 首帧模式

```bash
curl -X POST https://jubianai.vercel.app/api/v1/video/generate \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "jimeng",
    "mode": "first_frame",
    "image_base64": "base64_encoded_image_data",
    "duration": 5,
    "prompt": "可选提示词",
    "jimeng_access_key_id": "你的即梦AccessKeyId",
    "jimeng_secret_access_key": "你的即梦SecretAccessKey"
  }'
```

#### 首尾帧模式

```bash
curl -X POST https://jubianai.vercel.app/api/v1/video/generate \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "jimeng",
    "mode": "first_last_frame",
    "start_image_base64": "base64_encoded_start_image",
    "end_image_base64": "base64_encoded_end_image",
    "duration": 5,
    "prompt": "可选提示词",
    "jimeng_access_key_id": "你的即梦AccessKeyId",
    "jimeng_secret_access_key": "你的即梦SecretAccessKey"
  }'
```

## 技术实现

### 签名算法

使用火山引擎的 Signature V4 签名算法，具体实现见 `jubianai/backend/jimeng_client.py` 中的 `generate_volcengine_signature()` 函数。

### API 端点

- 默认端点：`https://visual.volcengineapi.com`
- 区域：`cn-north-1`
- 服务：`cv`

### 请求格式

根据即梦AI文档，请求使用以下格式：

- **req_key**: `jimeng_video_gen_v3_720p`（720P 图生视频）
- **image**: Base64 编码的图片（首帧模式）
- **start_image**: Base64 编码的首帧图片（首尾帧模式）
- **end_image**: Base64 编码的尾帧图片（首尾帧模式）
- **duration**: 视频时长（秒）

## 注意事项

1. **图片格式**：支持 PNG、JPG、JPEG 格式
2. **Base64 编码**：图片需要转换为 Base64 编码后传输
3. **API 密钥**：请妥善保管 API 密钥，不要泄露
4. **签名算法**：如果遇到签名错误，可能需要根据实际 API 文档调整签名算法
5. **API 路径**：如果实际 API 路径不同，需要在 `jimeng_client.py` 中修改 `uri` 变量

## 错误处理

如果遇到以下错误：

- **签名错误**：检查 API 密钥是否正确，签名算法是否符合要求
- **图片格式错误**：确保图片是 PNG、JPG 或 JPEG 格式
- **API 端点错误**：检查 `JIMENG_API_ENDPOINT` 环境变量是否正确
- **任务查询失败**：检查任务 ID 是否正确，API 密钥是否有效
