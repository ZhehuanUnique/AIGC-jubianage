# 资产管理功能说明

## 已实现功能

### 1. 资产上传
- ✅ 支持图片文件上传（JPG, PNG, GIF, WebP）
- ✅ 基于文件名的自动分类
- ✅ 文件名格式：`人物名-视图类型.扩展名`
  - 示例：`小明-正视图.jpg`、`小美-侧视图.png`
- ✅ 自动提取人物名称和视图类型

### 2. 资产分类和预览
- ✅ 按人物自动分组显示
- ✅ 每个分组下显示该人物的所有视图
- ✅ 图片预览功能
- ✅ 删除资产功能

### 3. 后端 API
- ✅ `/api/v1/assets/upload` - 上传资产
- ✅ `/api/v1/assets/list` - 获取所有资产（按人物分组）
- ✅ `/api/v1/assets/characters` - 获取人物列表
- ✅ `/api/v1/assets/{filename}` - 获取资产文件
- ✅ `/api/v1/assets/{filename}` (DELETE) - 删除资产

## 使用方法

### 上传资产

1. 进入"资产管理"标签页
2. 准备图片文件，文件名格式：`人物名-视图类型.扩展名`
   - 例如：`小明-正视图.jpg`
   - 例如：`小明-侧视图.png`
   - 例如：`小美-正视图.jpg`
3. 点击"选择图片文件"，选择一个或多个文件
4. 点击"上传资产"按钮
5. 系统会自动：
   - 解析文件名，提取人物名和视图类型
   - 保存文件到 `assets/` 目录
   - 创建元数据记录
   - 按人物分组显示

### 查看资产

在"资产管理"标签页的右侧，可以看到：
- 按人物分组的所有资产
- 每个资产显示为缩略图
- 显示视图类型（正视图、侧视图等）
- 可以删除不需要的资产

## 文件存储结构

```
jubianai/
├── assets/                    # 资产存储目录
│   ├── 小明-正视图_20231220_183000.jpg
│   ├── 小明-侧视图_20231220_183001.png
│   ├── 小美-正视图_20231220_183002.jpg
│   └── metadata.json          # 元数据文件
```

## 元数据格式

```json
{
  "assets": [
    {
      "filename": "小明-正视图.jpg",
      "character_name": "小明",
      "view_type": "正视图",
      "file_path": "assets/小明-正视图_20231220_183000.jpg",
      "upload_time": "2023-12-20T18:30:00",
      "file_size": 102400
    }
  ]
}
```

## 技术栈建议

如果要实现知识库功能，建议学习：

1. **多模态大模型**
   - CLIP（图像-文本匹配）
   - 多模态 LLM（如 GPT-4V、LLaVA）

2. **向量数据库**
   - Milvus（开源，性能好）
   - Chroma（轻量级，易用）
   - Pinecone（云服务）

3. **RAG（检索增强生成）**
   - LangChain（RAG 框架）
   - LlamaIndex（数据索引和检索）

4. **自然语言处理**
   - 命名实体识别（NER）识别人物
   - 文本分析和场景提取
