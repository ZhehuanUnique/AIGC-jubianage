## 流动式横向播放（丝滑减速停住 / 丝滑恢复）

这是一个**零依赖**的静态页面 Demo：无限横向循环滚动，鼠标悬停会**缓动减速到暂停**，移开后**缓动恢复到初始速度**。

### 移动端优化
- 移动端卡片尺寸优化（140px）
- 背景视频居中显示
- 卡片间距优化（10px）

### 运行方式

- **直接打开**：双击 `index.html`（大多数浏览器可正常运行）
- **建议本地静态服务器**（更接近线上环境）：
  - 若你有 Node.js：

```bash
npx --yes serve .
```

然后打开命令行输出的本地地址即可。

### 可调参数

在 `main.js` 的 `setupMarquee()` 中调整：
- `baseSpeedPxPerSec`：初始速度（px/s）
- `smoothness`：缓动强度（越大越快贴合目标速度）
